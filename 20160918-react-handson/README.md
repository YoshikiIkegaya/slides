## インストール

**※ Mac前提です**

【必要なもの】
+ Xcode
+ Commnad Line Tools
+ Homebrew

---

## Nodejsのインストール

**Nodeのバージョンは4以上にしてください**

```sh
$ brew update
$ brew install node
```

---

## Reactのインストール

```sh
$ npm install -g create-react-app
```

---

## 雛形の作成

```sh
$ create-react-app myapp
$ cd myapp
$ npm start
```

`http://localhost:3000/`にサンプルが表示される
